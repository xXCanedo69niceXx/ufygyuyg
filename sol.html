Solving Quadratic Equations using Factoring

1.) x² - 4x - 21 = 0
= (X - 7)(X + 3) = 0
- Get the factors of 21 that would make 4 possible
= X - 7 = 0
= X + 3 = 0
- Transpose to get the answers
= [X = 7][X = -3]

2.) x² + 5x - 36 = 0
=(X - 4)(X + 9)
- Get the factors of 36 that would make up 5 possible
= X - 4 = 0
= X + 9 = 0
- Transpose to get the answers
= [X = 4][X = -9]

3.) x² + 22x - 121 = 0
= (X - 11)(X - 11)
- Get the factors of 121 that would make up 22 possible
= X - 11 = 0
= X - 11 = 0
- Transpose to get the answers
= [X = 11][X = 11]

4.) x² - 2X - 15 = 0
= (X - 5)(X + 3)
- Get the factors of 15 that would make up 2 possible
= X - 5 = 0
= X + 3 = 0
- Transpose to get the answers
= [X = 5][X = -3]

5.) x² + X - 12 = 0
= (X - 4)(X = -9) - Get factors of 12 that would make X possible
= X - 4 = 0
= X + 3 = 0
- Transpose to get the answers
= [X = -4][X = 3]
